# Computer-Graphic
Лабораторная работа №2
Программа реализована на языке С#, использовались WinForms.
Предусмотрено
1) Выбор необходимой папки с изображениями
В результате работы программы, пользователь получает детальную информацию о каждом изображении, которая отображается автоматически в таблице.
Для решения задачи с помощью менеджера NuGet была установлена библиотека ImageFormats by Dmitry Brant.
Для анализа быстродействия по завершении работы выводится информация о затраченном времени. Данные доставались с помощью либо стандартных инструментов C#, либо с использованием методов из сторонней библиотеки.
Время выполнения на папке размером 2.36гб: 8.238 секунд
Данная папка может быть предоставлена по требованию для тестирования приложения.
